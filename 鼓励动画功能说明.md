# 鼓励动画功能说明

## 功能概述

当用户点击卡片查看答案时，屏幕右上角会显示一个可爱的鼓励动画，随机显示鼓励语，增加学习的趣味性和成就感。

## 视觉效果

### 动画特点

- **位置**：固定在屏幕右上角（top-20 right-4）
- **渐变背景**：从靛蓝到紫色到粉色的渐变色
- **光晕效果**：背景有模糊的光晕，营造梦幻感
- **装饰元素**：
  - 右上角旋转的 ✨ 星星
  - 左下角跳动的 🎉 彩带
- **动画效果**：
  - 轻柔的上下浮动（bounce-gentle）
  - 淡入淡出过渡
  - 缩放效果

### 显示时长

- 显示 2.5 秒
- 淡出 0.5 秒
- 总计 3 秒后完全消失

## 触发条件

- 用户点击卡片查看答案时触发
- 只在首次查看答案时触发（已查看过的卡片不会重复触发）
- 适用于专注模式和列表模式

## 技术实现

### 组件结构

```
EncouragementToast
├── 主卡片（渐变边框）
│   └── 白色内容区（鼓励文字）
├── 光晕效果（模糊背景）
└── 装饰元素（星星、彩带）
```

### 动画定义

在 `app/globals.css` 中定义了三个自定义动画：

1. **bounce-gentle**：轻柔的上下浮动

   - 持续时间：2 秒
   - 效果：上下移动 10px

2. **spin-slow**：缓慢旋转

   - 持续时间：3 秒
   - 效果：360 度旋转

3. **bounce-slow**：缓慢跳动
   - 持续时间：2 秒
   - 效果：上下移动 5px

### 状态管理

- `showEncouragement`：控制动画显示/隐藏
- 通过 `onShowEncouragement` 回调触发
- 自动计时器控制显示时长

## 自定义配置

### 修改鼓励语

编辑 `components/EncouragementToast.tsx` 中的 `ENCOURAGEMENT_MESSAGES` 数组：

```typescript
const ENCOURAGEMENT_MESSAGES = [
  "你很棒哦！ 🌟",
  "干得漂亮！ 👏",
  // 添加更多鼓励语...
];
```

### 修改显示时长

在 `EncouragementToast.tsx` 的 `useEffect` 中调整：

```typescript
// 1.7秒后开始淡出
const hideTimer = setTimeout(() => {
  setIsVisible(false);
}, 1700); // 修改这个值

// 2秒后完全隐藏
const removeTimer = setTimeout(() => {
  setIsShowing(false);
  onComplete();
}, 2000); // 修改这个值
```

### 修改位置

在 `EncouragementToast.tsx` 中修改位置类名：

```typescript
// 右上角（默认）
className = "fixed top-20 right-4 z-50";

// 左上角
className = "fixed top-20 left-4 z-50";

// 居中顶部
className = "fixed top-20 left-1/2 -translate-x-1/2 z-50";
```

### 修改颜色主题

修改渐变色：

```typescript
// 当前：靛蓝-紫色-粉色
className = "bg-gradient-to-r from-indigo-500 via-purple-500 to-pink-500";

// 绿色主题
className = "bg-gradient-to-r from-green-500 via-emerald-500 to-teal-500";

// 橙色主题
className = "bg-gradient-to-r from-orange-500 via-amber-500 to-yellow-500";
```

## 用户体验

### 优点

- ✅ 增加学习的趣味性
- ✅ 提供即时的正向反馈
- ✅ 鼓励用户继续学习
- ✅ 随机性增加新鲜感
- ✅ 不干扰主要内容

### 注意事项

- 动画不会阻塞用户操作
- 自动消失，无需手动关闭
- 不会重复触发（已查看的卡片）
- 移动端和桌面端都适配
