# Flash 闪现效果说明

## 改进内容

将鼓励动画从 2 秒改为更快的 Flash 闪现效果。

## 时间调整

### 改进前

```
0ms    - 动画开始（淡入）
1700ms - 开始淡出
2000ms - 完全消失
总时长: 2秒
```

### 改进后

```
0ms    - 动画开始（快速淡入）
1000ms - 开始淡出
1300ms - 完全消失
总时长: 1.3秒
```

## 动画速度调整

### 改进前

- 过渡时长: 500ms
- 缩放: 95% → 100%
- 效果: 较慢、平滑

### 改进后

- 过渡时长: 300ms（快 40%）
- 缩放: 90% → 100%（更明显）
- 效果: 快速、有冲击力

## Flash 效果特点

1. **快速出现** (0-300ms)

   - 快速淡入
   - 快速放大
   - 快速上移

2. **短暂停留** (300-1000ms)

   - 停留 0.7 秒
   - 足够阅读鼓励语
   - 不会过度干扰

3. **快速消失** (1000-1300ms)
   - 快速淡出
   - 快速缩小
   - 快速下移

## 技术实现

### CSS 过渡

```css
transition-all duration-300  /* 从 duration-500 改为 duration-300 */
```

### 缩放效果

```css
scale-90  /* 从 scale-95 改为 scale-90，更明显 */
```

### 时间控制

```typescript
setTimeout(() => setIsVisible(false), 1000); // 从 1700 改为 1000
setTimeout(() => setIsShowing(false), 1300); // 从 2000 改为 1300
```

## 如果需要更快

如果觉得还不够快，可以进一步调整：

### 超快版本（0.8 秒）

```typescript
setTimeout(() => setIsVisible(false), 600); // 0.6秒后淡出
setTimeout(() => setIsShowing(false), 800); // 0.8秒后消失
```

### 极速版本（0.5 秒）

```typescript
setTimeout(() => setIsVisible(false), 300); // 0.3秒后淡出
setTimeout(() => setIsVisible(false), 500); // 0.5秒后消失
```

## 建议

当前的 1.3 秒是一个很好的平衡：

- 足够快，不会打断学习
- 足够慢，能看清鼓励语
- Flash 效果明显
- 用户体验良好

如果需要调整，建议在 1-1.5 秒之间。
