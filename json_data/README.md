# 数据集文件夹

这个文件夹用于存放所有的学习数据集。

## 如何添加新数据集

1. 在此文件夹中创建新的 JSON 文件
2. 按照以下格式编写内容
3. 刷新页面即可看到新数据集

## JSON 文件格式

```json
[
  {
    "id": 1,
    "question": "问题内容",
    "answer": "答案内容",
    "author": "作者名（可选，可为 null）",
    "category": "分类（可选）"
  },
  {
    "id": 2,
    "question": "另一个问题",
    "answer": "另一个答案",
    "author": null,
    "category": "分类"
  }
]
```

## 字段说明

- `id`: 数字类型，必填，在同一文件中必须唯一
- `question`: 字符串类型，必填，问题内容
- `answer`: 字符串类型，必填，答案内容
- `author`: 字符串或 null，可选，作者名称
- `category`: 字符串类型，可选，分类标签

## 示例数据集

- `quotes.json` - 智慧名言（英文励志名言）
- `poems.json` - 古诗词（中文诗词名句）
- `example.json` - 示例数据集

## 注意事项

1. 确保 JSON 格式正确
2. 每个文件中的 id 必须唯一
3. 文件名建议使用英文小写字母和下划线
4. 添加或删除文件后刷新页面即可生效
