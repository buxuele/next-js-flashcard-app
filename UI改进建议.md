# UI 改进建议

## 当前问题分析

### 1. 侧边栏显示问题 ⚠️

**问题**: 当前侧边栏显示的是处理后的名称（去掉了"读书笔记--"前缀），导致：

- "读书笔记--丰臣秀吉 1" 显示为 "丰臣秀吉 1"
- "读书笔记--德川家康 4" 显示为 "德川家康 4"
- 用户无法看到完整的文件名
- 不同系列的笔记难以区分

**建议**: 显示完整的文件名（不含扩展名）

### 2. 数据集分类问题

**问题**: 21 个数据集平铺显示，没有分类：

- 滚动列表很长
- 难以快速找到想要的内容
- 缺乏组织结构

**建议**: 添加分类功能

### 3. 搜索/筛选功能缺失

**问题**: 数据集多了之后，查找困难

**建议**: 添加搜索框

## 详细改进方案

### 优先级 1: 显示完整文件名 ⭐⭐⭐

#### 改进前

```
丰臣秀吉 1
德川家康 4
李斯 1
```

#### 改进后

```
读书笔记--丰臣秀吉 1
读书笔记--德川家康 4
读书笔记--李斯 1
```

#### 实现方式

**选项 A**: 完全显示原始文件名

```typescript
// 不做任何处理，直接使用文件名
displayName = nameWithoutExt;
```

**选项 B**: 保留前缀，但美化显示

```typescript
// 将 "--" 替换为 " · "
displayName = nameWithoutExt.replace(/--/g, " · ");
// 结果: "读书笔记 · 丰臣秀吉 1"
```

**选项 C**: 显示文件名 + 副标题

```typescript
// 主标题: 书名
// 副标题: 系列信息
<div className="font-medium text-sm">丰臣秀吉 1</div>
<div className="text-xs opacity-60">读书笔记系列</div>
<div className="text-xs opacity-75">{dataSet.data.length} 张卡片</div>
```

**推荐**: 选项 A（最简单直接）或选项 B（更美观）

在 JSON 文件中添加 category 字段：

```json

  "items": [...]
}
```

**推荐**: 方案 A（自动分类，无需修改数据）

## 其他 UI 改进建议

### 2. 学习模式增强

**当前**: 专注模式 + 列表模式
**建议**:

- 添加"复习模式"（只显示未掌握的）
- 添加"随机模式"（打乱顺序）

**建议**: 添加学习统计

### 4. 快捷键增强

**当前**: ← → 切换卡片
**建议**:

- Space: 翻转卡片
- M: 标记为已掌握
- S: 收藏卡片
- R: 重置当前数据集
- /: 聚焦搜索框

### 5. 导出功能

**建议**:

- 导出学习进度（JSON）

- 分享学习成果

### 6. 主题切换

**当前**: 仅暗色主题
**建议**:

- 亮色主题
- 自动跟随系统
- 自定义主题色
