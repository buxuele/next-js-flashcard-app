# 侧边栏折叠功能说明

## 功能特性

### 桌面端（大屏幕）

- **折叠/展开按钮**：位于侧边栏右侧的圆形按钮
- **展开状态**（默认）：
  - 宽度 256px (w-64)
  - 显示完整的数据集名称和卡片数量
  - 显示底部提示文字
- **折叠状态**：
  - 宽度 64px (w-16)
  - 只显示数据集名称首字母图标
  - 鼠标悬停时显示完整信息的浮动提示框
  - 顶部显示数据集图标

### 移动端（小屏幕）

- **汉堡菜单按钮**：固定在左上角
- **侧边栏**：默认隐藏，点击按钮后从左侧滑出
- **遮罩层**：打开侧边栏时显示半透明黑色遮罩
- **自动关闭**：选择数据集后自动关闭侧边栏

## 技术实现

### 响应式设计

- 使用 Tailwind CSS 的 `lg:` 断点（1024px）
- 移动端和桌面端有不同的交互方式
- 平滑的过渡动画（transition-all duration-300）

### 状态管理

- `isOpen` 状态控制展开/折叠
- 桌面端默认展开
- 移动端默认折叠

### 交互细节

1. **折叠按钮**：

   - 圆形设计，带边框和阴影
   - 箭头图标根据状态旋转 180 度
   - 悬停时背景色变化

2. **数据集按钮**：

   - 展开时显示完整信息
   - 折叠时显示首字母圆形图标
   - 悬停时显示浮动提示框

3. **动画效果**：
   - 宽度变化平滑过渡
   - 箭头旋转动画
   - 内容淡入淡出

## 使用方法

### 桌面端

1. 点击侧边栏右侧的圆形按钮折叠/展开
2. 折叠后鼠标悬停在数据集图标上查看详情
3. 点击数据集切换学习内容

### 移动端

1. 点击左上角的汉堡菜单按钮打开侧边栏
2. 选择数据集后自动关闭
3. 点击遮罩层也可关闭侧边栏

## 自定义配置

如需修改默认状态，编辑 `app/page.tsx`：

```typescript
// 默认展开
const [sidebarOpen, setSidebarOpen] = useState(true);

// 默认折叠
const [sidebarOpen, setSidebarOpen] = useState(false);
```

## 样式调整

在 `components/Sidebar.tsx` 中可以调整：

- 展开宽度：`w-64` (256px)
- 折叠宽度：`lg:w-16` (64px)
- 动画时长：`duration-300` (300ms)
- 按钮位置：`-right-3 top-6`
