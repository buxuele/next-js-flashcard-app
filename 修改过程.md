# 修改过程

## 需求：支持多数据集

此时只有 2 个数据，如果我有 10 个、100 个呢？

### 要求

- 把数据文件都放到 json_data 文件夹 ✅
- UI 界面在左边增加一个侧边栏，显示每个数据集的文件名，用户选一个可以进行学习 ✅

### 实现说明

#### 1. 动态数据集加载

- 创建了 `json_data` 文件夹存放所有数据文件
- 添加了 `/api/datasets` API 路由，自动扫描并加载所有 JSON 文件
- 支持无限扩展，只需添加 JSON 文件即可

#### 2. 左侧边栏

- 新增 `Sidebar.tsx` 组件
- 显示所有可用的数据集及卡片数量
- 支持响应式设计（移动端可折叠）
- 当前选中的数据集高亮显示

#### 3. 架构改进

- 移除了硬编码的数据集枚举（DataSet enum）
- 使用动态 ID 系统替代固定枚举
- 简化了状态管理逻辑
- 删除了 AI 生成功能（可根据需要恢复）
